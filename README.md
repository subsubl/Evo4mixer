# Evo4mixer

Advanced Linux mixer for Audient EVO 4 audio interfaces.

## Features
- ALSA-based control (no audio playback conflict)
- RME-inspired user interface
- Real-time volume monitoring (WIP)

## Getting Started
1. Install [Tauri](https://tauri.app/v2/guides/getting-started/setup/linux/)
2. Install [Trunk](https://trunkrs.dev/)
3. `npm install`
4. `npm run tauri dev`
